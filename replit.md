# PF2 Character Sheet

Flask-приложение для создания и управления листами персонажей Pathfinder 2e Remastered.

## Стек

- **Backend**: Python 3.11 + Flask 2.3.3
- **БД**: SQLite через Flask-SQLAlchemy 3.1.1
- **Frontend**: Jinja2 шаблоны + CSS

## Структура

```
├── app.py              # Flask-приложение, маршруты
├── models.py           # Модель Character (SQLAlchemy)
├── requirements.txt
├── templates/
│   ├── base.html
│   ├── index.html          # Список персонажей
│   ├── create_character.html  # Форма создания/редактирования
│   └── character.html      # Просмотр листа персонажа
└── static/
    └── style.css
```

## Запуск

```bash
python app.py
```

Сервер запускается на порту 5000.

## Функциональность

- Создание, просмотр, редактирование и удаление персонажей
- 6 основных характеристик (Сила, Ловкость, Телосложение, Интеллект, Мудрость, Харизма)
- Автоматический расчёт модификаторов характеристик
- 16 навыков с уровнями владения (Необучен / Обучен / Эксперт / Мастер / Легендарный)
- Автоматический расчёт бонусов навыков по правилам PF2e
- Боевые параметры: ХП, Класс Брони
