# PF2e Character Sheet

Flask-приложение для создания и управления листами персонажей Pathfinder 2e Remastered (на русском языке).

Главный источник информации по правилам: **pf2.ru**

## Стек

- **Backend**: Python 3.11 + Flask 2.3.3 + Gunicorn
- **БД**: SQLite через Flask-SQLAlchemy 3.1.1
- **Frontend**: Jinja2 + Vanilla JS + CSS (без сторонних фреймворков)

## Структура

```
├── app.py                  # Flask-приложение, маршруты, DB-миграция
├── models.py               # Модель Character (SQLAlchemy)
├── requirements.txt
├── templates/
│   ├── base.html
│   ├── index.html              # Список персонажей
│   ├── create_character.html   # Форма создания/редактирования
│   ├── character.html          # Просмотр листа персонажа
│   └── import.html             # Импорт из JSON
└── static/
    ├── style.css
    ├── js/character.js         # Клиентская логика
    └── uploads/                # Загруженные аватары
```

## Запуск (dev)

```bash
python app.py
```

Prod через gunicorn:
```bash
gunicorn --bind=0.0.0.0:5000 --reuse-port app:app
```

## Функциональность

- Основная информация: имя, класс (21 класс PF2e), уровень, **народ**, **происхождение**, опыт, языки
- Модификаторы характеристик (PF2e Remastered — без значений, только модификатор)
- Боевые параметры: ХП с интерактивными ±кнопками, КБ, скорость
- Испытания (Стойкость/Реакция/Воля) с владением и автоматическим расчётом бонуса
- Восприятие с владением
- **17 навыков** (PF2e Remastered): Акробатика, Атлетика, Воровство, Выживание, Дипломатия, Запугивание, Исполнение, Медицина, **Мистицизм**, Обман, Общество, Оккультизм, Природа, Религия, Ремесло, Скрытность + **Знание** (с темой)
- **Владение доспехами**: Без брони / Лёгкий / Средний / Тяжёлый
- **Владение оружием**: Безоружные / Простое / Воинское / Экзотическое
- Классовая сложность (КС) и бонус атаки заклинаниями
- Черты: **общие, навыков, народа, происхождения, классовые** — динамически добавляются
- Классовые умения — динамически добавляются
- Заклинания: традиция (**Мистические/Сакральные/Оккультные/Первобытные**), тип (подготовленные/спонтанные), слоты по рангам 1-10, заговоры, заклинания
- **Фокальные заклинания** с количеством фокальных очков
- Инвентарь: название, количество, вес, цена, экипировка
- Аватар персонажа (загрузка изображения, превью)
- Экспорт в JSON / импорт из JSON
- Дизайн в стиле Pathfinder (тёмно-красный + золото)

## Архитектура данных

Черты, умения, заклинания и инвентарь хранятся как JSON-строки в полях Character.
Фокальные заклинания и фокальные очки хранятся внутри `spells_json`.

При запуске `app.py` автоматически выполняется SQLite-миграция для добавления новых колонок к существующим персонажам (безопасно — колонки пропускаются если уже существуют).

## Известные проблемы

- БД сбрасывается при перезапуске контейнера. Планируется перенос на облачное хранилище.
